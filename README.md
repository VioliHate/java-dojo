# 🥋 Java Dojo

> My personal training ground to get back in shape with Java

### Welcome to the Dojo! 🏯

Hi there!  

This is my **Java Dojo** — a personal space I created to sharpen my Java skills after years of not practicing. Not starting from zero, but definitely ready to rebuild strong foundations and level up.

Here I train consistently, exercise after exercise, with discipline and (most importantly) **fun**.

---

## 🎯 Purpose

To get back into shape with Java through daily practical exercises, clean code, and steady improvement.

---

## 🧘‍♂️ Rules of the Dojo

1. **Fun comes first** 
2. Learn **at least one new thing** every day
3. Always correct and refine the exercises after they work
4. Write **clean, readable code** and comment where it makes sense
5. Solve problems using logic first, even if it takes longer
6. Internet is a great place… but I only consult it **after** the training session ends

## 🛠️ How to Run

```bash
# Clone the repository
git clone https://github.com/YOURUSERNAME/java-dojo.git

# Navigate to a daily exercise
cd java-dojo/main/violihate/javadojo/day02oop

# Compile
javac *.java

# Run (example)
java E02_StudentManagement
```
